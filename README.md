# Practica-6-71486

Para construir una barda es necesario determinar con precisión la cantidad de bloques requeridos y el costo total del proyecto. 
Este proceso comienza calculando el área del muro a partir de sus dimensiones (alto y largo). 
Una vez obtenida el área total, se divide entre el área individual de un bloque de concreto para estimar la cantidad aproximada de piezas necesarias.

Además, se considera un 5% adicional por concepto de desperdicio, roturas o cortes que suelen presentarse durante la construcción. 
Esta cantidad extra se suma al cálculo original y posteriormente se redondea hacia arriba para asegurar que no falten materiales durante la obra.

Con el número total de bloques determinado, se multiplica por el precio unitario de cada pieza, obteniendo así el costo final estimado.

El programa en Python automatiza todo este procedimiento, permitiendo ingresar las dimensiones del muro y el costo del bloque para generar una estimación rápida, confiable y fácil de interpretar. 
De esta forma, se evita realizar cálculos manuales propensos a errores y se facilita la planificación del presupuesto en proyectos de albañilería o construcción.
